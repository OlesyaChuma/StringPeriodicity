# StringPeriodicity

Проект демонстрирует проверку периодичности строки по заданному числу `k` с использованием 
алгоритма Кнута–Морриса–Пратта (КМП) и префикс-функции.

## Описание задачи

Строка считается **k-периодичной**, если она полностью состоит из повторений 
подстроки длины `k`.  
Например:

- `abcabcabcabc` при `k = 3` → периодична (`abc` повторяется 4 раза)
- `ababab` при `k = 2` → периодична (`ab` повторяется 3 раза)
- `abac` при `k = 2` → НЕ периодична

Проект реализует класс `StringPeriodicity`, который содержит:

- вычисление префикс-функции (`PrefixFunction`)
- проверку k-периодичности (`IsKPeriodic`)

Алгоритм работает за **O(n)** времени и использует **префикс-функцию КМП**, 
что является оптимальным способом определения периода строки.

---

## Используемый алгоритм (КМП + префикс-функция)

Для строки длины `n` вычисляется π-массив.

Если строка периодична, то период равен:

k = n - π[n - 1]

## Реализованные функции

### `PrefixFunction(const std::string& s)`
Вычисляет префикс-функцию КМП для строки `s`.

### `IsKPeriodic(const std::string& s, int k)`
Проверяет, является ли строка `s` k-периодичной.

## Пример использования

```cpp
#include "StringPeriodicity.h"

std::string s = "abcabcabcabc";
int k = 3;

if (StringPeriodicity::IsKPeriodic(s, k))
{
    std::cout << "Строка периодична" << std::endl;
}
else
{
    std::cout << "Строка НЕ периодична" << std::endl;
}
